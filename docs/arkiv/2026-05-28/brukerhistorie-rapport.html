Hva er brukerhistorietesting?

Brukerhistorietester verifiserer at konkrete brukerscenarioer fungerer i applikasjonen. Hver test representerer én brukerhistorie med akseptansekriterier som automatisk verifiseres mot testmiljøet.

Hva testes
  • 📋 Navigasjon og oversikt over utlysninger
  • 🔍 Søk etter tilskuddsordninger
  • 👤 Innlogget brukers «Min side»
  • 🔙 Tilbake-navigasjon og F5-refresh
  • ♿ Skiplink og hjelpemiddelteknologi
Hva måles
  • Antall bestatte akseptansekriterier
  • Antall feilende akseptansekriterier
  • Feilmelding og stack trace ved feil
  • Skjermbilder ved feiling
Verktøy og metode
  • @playwright/test (testramme)
  • Innlogging via ID-porten TestID
  • Kjøres mot testmiljøet
  • Kjøres daglig og ved manuell trigger
